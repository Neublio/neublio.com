# Neublio landing page

[![Netlify Status](https://api.netlify.com/api/v1/badges/f4ae90ef-c7b9-4304-9fd6-b2d7133e0321/deploy-status)](https://app.netlify.com/projects/neublio/deploys)

This is the landing page repo for Neublio, an assistant tool and web service for knowledge work.

This webpage runs on Gatsby, Reactjs, TailwindCSS and Material-UI.

## To run on dev mode

`gatsby develop`

## To build web site

`gatsby build`
